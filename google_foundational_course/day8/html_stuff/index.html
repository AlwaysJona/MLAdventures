Day 8

Simplicity

Overcrossing could be a problem, if our data is basically linear with some noise, giving too many feature crosses might make the model overfit to the noise.

So we want as simple as a model as possible, and if the weights of some features are way bigger the others, that might suggest an overcomplicated model. To avoid this our model needs some kind of regularization, to minimize complexity.

L2 Regularization

L2 Regularization consists of making the model minimize not only the loss but also the sum of the weights squared, times some constant that depends on the data, if training data and test data are very similar, the constant will be very low.

Logistic Regression

For some data we might only need a probability, for example for a coin toss. This probability is going to help us make predictions. To do that we use linear logistic regression that has values only between 0 and 1, a sigmoid function has those characteristics.

Loss and regularization

For linear regression we use squared losses, for logistic regression we uses Log Loss. Regularization is very important in logistic regression, since the sigmoid tends to zero or one at plus or minus infinity, the model will push the weights so high, or so low, just to get there, and in no time we would have an overfit model. To avoid this we can use both L2 regularization or stopping early.
