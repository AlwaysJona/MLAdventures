Programming excercise on validation set

The program run was basically the same as in the previous excercise, this time there was a division between train_df and test_df. And i added an argument to the train_model function which was validation_split effectively dividing the train_df in 2. When you get your results you need to make sure that when the model converges the training rmse and validation rmse are the same otherwise there is some problem either in the data or the model. In this case was the data, which was not properly shuffled. After shuffling the two rmse were about the same. Finally you validate your model with test_df and if the returned rmse is about the same as those of the training and validation then you're good.
